# FlareLax Egg

This is the ultimate Minecraft egg to fulfill a hoster's need for hibernation and resource management!

# Features 

1) Non-bypassable Hibernation and auto-shutdown
2) Shutdown in 5 minutes if no players are online
3) Dedicated strictly to PaperMC to ensure optimal security and resource distribution on FlareLax
4) Completely normal server performance when players are online
5) Auto-starts the server dynamically when a player attempts to join via Multiplayer menu!
